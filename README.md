# School Management System

## Project Description
The " School Management System " is designed to simplify school management digitally. The system allows managing students, teachers, courses, and classrooms in an organized and easy way. 

## Features

- Student Management: Add, delete and view student data.  
- Teacher Management: Add, delete and view teacher data.  
- Course Management: Create and delete courses.  
- Enroll students in courses  
- Classroom Management  

## Technologies Used
- C# with Windows Forms (GUI).  
- database integration.
